# Application Running 🚀

The Local RAG Chatbot is up and running.

## 🔗 Access Links
- **Frontend (Chat Interface):** [http://localhost:8501](http://localhost:8501)
- **Backend (API Documentation):** [http://localhost:8000/docs](http://localhost:8000/docs)

## ⚡ Quick Start
1.  Open the **Frontend** link.
2.  **Upload** your PDF/TXT Documents.
3.  Click **"Ingest"**.
4.  Start **Chatting**!

> **Note**: Your terminal windows will show the logs. To stop the app, press `Ctrl+C` in the respective terminals.
